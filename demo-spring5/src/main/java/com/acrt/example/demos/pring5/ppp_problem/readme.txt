问题：
1.@Resource默认先按name还是先按type装配？
    答：默认先按name，name匹配不到再按type；如果填写name（或者type也有）只会按name装配
2.final修饰的类能被cglib代理吗？
    答：不能，cglib代理的原理是继承目标类，如果目标类不能被继承自然会报错
3.切入点表达式对重载的方法如何区分？
    答：切入点表达式格式：execution([修饰符] 返回值类型 包名.类名.方法名(参数))、
        参数的格式可以指定类型：如：方法名(java.lang.String, java.*.String)
          *：匹配任何数量字符；
          ..：匹配任何数量字符的重复，如任何数量子包、任何数量参数。
          +：匹配指定类型的子类型；仅能作为后缀放在类型模式后边。
